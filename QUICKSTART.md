# 快速开始指南

## 5分钟快速上手

### 1️⃣ 本地运行

**方法A: 直接打开HTML文件**
- 双击 `index.html` 文件
- 在浏览器中打开

**方法B: 使用本地服务器 (推荐)**

使用Python (推荐,Python 3):
```bash
python -m http.server 8000
```

或者使用Python 2:
```bash
python -m SimpleHTTPServer 8000
```

使用Node.js:
```bash
npx http-server -p 8000
```

使用PHP:
```bash
php -S localhost:8000
```

然后访问: http://localhost:8000

### 2️⃣ 使用计算器

#### 步骤1: 选择品质
在"品质等级"下拉菜单中选择你想要使用的皮肤品质:
- 消费级 → 产出工业级
- 工业级 → 产出军规级
- 军规级 → 产出受限
- 受限 → 产出保密
- 保密 → 产出隐秘

#### 步骤2: 选择武器箱
在"武器箱/收藏"下拉菜单中选择一个武器箱,例如:
- 美梦与噩梦武器箱
- 革命武器箱
- 反冲武器箱
- 千瓦武器箱

#### 步骤3: 搜索皮肤
点击"搜索可用皮肤"按钮,系统将:
- 加载该武器箱中对应品质的所有皮肤
- 获取每个皮肤的当前市场价格
- 显示在可用皮肤列表中

#### 步骤4: 选择10个皮肤
点击皮肤卡片将其添加到已选择列表:
- 必须选择恰好10个皮肤
- 所有皮肤必须是相同品质
- 可以点击已选择的皮肤来取消选择

#### 步骤5: 计算收益
当选择了10个皮肤后:
- "计算期望收益"按钮会变为可用
- 点击按钮开始计算
- 系统将显示详细的收益分析

### 3️⃣ 理解计算结果

计算器会显示以下信息:

#### 📊 收益汇总
- **总投入成本**: 10个输入皮肤的总价格
- **期望产出价值**: 根据概率计算的平均产出价值
- **期望收益**: 期望产出价值 - 总投入成本
- **投资回报率(ROI)**: (期望收益 / 总投入成本) × 100%

#### 🎁 可能的产出物品
表格显示所有可能产出的皮肤:
- **物品名称**: 可能产出的皮肤
- **品质**: 比输入高一级的品质
- **当前价格**: 该皮肤的市场价格
- **概率**: 产出该皮肤的概率
- **期望贡献**: 价格 × 概率 = 对期望值的贡献

### 4️⃣ 示例操作

**示例: 计算军规级汰换合同**

1. 选择品质: `军规级 (Mil-Spec)`
2. 选择武器箱: `美梦与噩梦武器箱`
3. 点击"搜索可用皮肤"
4. 选择10个军规级皮肤(建议选择价格较低的)
5. 点击"计算期望收益"
6. 查看结果:
   - 如果ROI > 0%: 期望盈利
   - 如果ROI < 0%: 期望亏损

**投资建议**:
- ✅ ROI > 20%: 非常好的投资机会
- ✅ ROI > 0%: 理论上可以盈利
- ⚠️ ROI < 0%: 不建议进行
- ⚠️ ROI < -20%: 很可能亏损

## 常见问题

### Q: 为什么价格数据不准确?
A: 当前版本使用模拟数据用于演示。要使用真实价格,需要配置Steam API。详见 `DEPLOYMENT.md` 中的API配置部分。

### Q: 可以选择不同武器箱的皮肤吗?
A: 可以!汰换合同支持混合来自不同武器箱的皮肤。产出物品会根据每个武器箱的皮肤数量按比例分配概率。

### Q: 如何提高盈利概率?
A: 
1. 选择价格低的输入皮肤
2. 选择产出高价皮肤的武器箱
3. 比较多个方案,选择ROI最高的
4. 关注市场价格波动

### Q: 磨损度如何选择?
A: 当前版本默认使用Field-Tested(略有磨损)。后续版本将支持自定义磨损度。

### Q: 计算结果可靠吗?
A: 期望值计算是准确的,但实际结果会有运气成分:
- 你可能运气好,产出高价物品
- 也可能运气差,产出低价物品
- 长期来看,结果会趋向期望值

### Q: 如何保存计算方案?
A: 当前版本不支持保存。建议截图保存结果,或使用浏览器的书签功能。

## 技巧和建议

### 💡 寻找盈利机会
1. **关注新武器箱**: 新箱子上市初期价格波动大,可能有套利机会
2. **比较多个品质**: 有时高品质汰换反而ROI更高
3. **考虑手续费**: Steam市场有约13%的交易手续费
4. **监控价格趋势**: 皮肤价格会随时间变化

### ⚠️ 风险警示
- 期望收益 ≠ 保证收益
- 实际结果可能与计算值偏差很大
- 市场价格实时波动
- 交易有手续费和延迟
- 投资需谨慎,不要投入超过承受能力的资金

### 📈 进阶策略
1. **批量操作**: 多次汰换可以平滑运气波动
2. **对冲风险**: 同时进行多个不同的汰换方案
3. **市场时机**: 在价格低谷买入,高峰卖出
4. **长期持有**: 某些稀有皮肤长期会升值
